# WEDM 线切割 G-Code 在线模拟器

帮助用户无需上机，无需下载任何“重的”APP，通过浏览器【编辑｜单步运行｜连续运行】G-Code，预览最终切割的效果。

## 特性一览

### 已经实现的特性

- [X] G-Code 在线编辑
- [X] 逐步运行你的代码
  - [X] 逐步运行时，高亮当前代码行
- [X] 连续运行你的代码
- [X] 可以缩放、拖动的画布
- [X] 在 Scale、代码变化时 自动重置画布
